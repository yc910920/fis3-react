### react project scaffolding
---


#### download project
`1. install node`

`2. git clone https://github.com/shanchenbaby/fis3-react.git`

`3. cd fis3-react`

`4. npm i fis3 -g`

`5. npm i`


#### use project
`1. npm run start ( 开启 local web server )`

`2. npm run dev ( 开启 dev mode )`


#### commit code
`1. npm run commit ( 源代码提交到 master )`

`2. npm run build ( 资源产出 )`
